WBEM Provider Data Sheet

Computer System Provider

This data sheet provides brief, very basic and the most up-to-date information available about the Computer System WBEM provider (implemented using the Pegasus C++ API).  This provider supports the CIM_ComputerSystem, CIM_UnitaryComputerSystem, and PG_ComputerSystem classes.  This document is not intended to replace manuals or man pages.   The intended audience of this document is client/provider developers and sophisticated end-users who want to use this provider. 1. Provider Overview

·         Description
The Computer System provider makes available basic computer system information such as computer name, status, and administrator contact information. This Computer System provider instruments the CIM_ComputerSystem, CIM_UnitaryComputerSystem, and PG_ComputerSystem classes.  The PG_ComputerSystem subclass of CIM_UnitaryComputerSystem adds the PrimaryOwnerPager, SecondaryOwnerName, SecondaryOwnerContact, SecondaryOwnerPager, SerialNumber, and IdentificationNumber properties.

      • PrimaryOwnerPager is the pager number for the primary system owner.
      • SecondaryOwnerName, SecondaryOwnerContact, and SecondaryOwnerPager are the name, phone number, and pager number of the system’s secondary owner respectively.
      • SerialNumber is the system’s serial number.
      • IdentificationNumber is the corporate asset number of the system.

This provider is for use by clients as part of a basic understanding of the identity of the Managed System on which it is running (typically a server or appliance).  The current implementation is for HP-UX only. 

·         Requirements
Implemented and tested on HP-UX 11i PA-RISC based systems.

·         Release History
11-Mar-2002 - initial release (via submittal to Pegasus open source community)

·         Supported Managed Resources
Managed systems (servers and appliances) running HP-UX.

·         Special Requirements & Dependencies
None.

2. Setting Up This Provider

·         Installing This Provider

This Computer System provider is available from the Pegasus open source site (CVS tree location: pegasus/src/ManagedSystem/ComputerSystem for the makefile, headers and source; pegasus/Schemas/pegasus/ManagedSystem/VER20/PG_ComputerSystem20.mof for the MOF file for the schema; and PG_ComputerSystem20R.mof for the provider registration). 

The provider executable is "libComputerSystemProvider.sl".  The provider is registered to support the “root/cimv2” namespace as an instance provider.  There are no special installation instructions; the provider will be installed by default in most Pegasus-based product releases.

·        Configuring This Provider
This provider does not have specific configuration adjustments that can be made (beyond standard WBEM support).

3. Using This Provider

·         Schema supported by this provider

This provider supports the CIM_ComputerSystem, CIM_UnitaryComputerSystem, and PG_ComputerSystem classes.  The following tables discuss the properties supported by the provider.  Note that while some properties in the table are marked with the “Write” qualifier, all properties are in fact read-only.  The “Write” qualifier is only intended as a hint that a given property is one that could be instrumented by a provider to be modifiable by a client application.

Table: CIM_ComputerSystem Properties

 Property Name

 Inherited or Local

Description (& data source)

Caption

Inherited from CIM_ManagedElement

Always returns string “Computer System”.

Description

Inherited from CIM_ManagedElement

Always returns string “This is the CIM_ComputerSystem object.”

InstallDate

Inherited from CIM_ManagedSystemElement

The date/time the kernel was created.  Obtained using the stat() system call on the /stand/vmunix file.

CreationClassName [Key]

Inherited from CIM_System

Always returns string “PG_ComputerSystem”.

Name [Key]

Inherited from CIM_ManagedSystemElement, overridden in CIM_System

The fully qualified host name returned by the name service through the gethostbyname() system call.  If a name service is not available, the provider supplies the hostname returned by the gethostname() system call.

Status

Inherited from CIM_ManagedSystemElement

Always returns 0 (“OK”).

NameFormat

Inherited from CIM_System, overridden in CIM_ComputerSystem

Always returns 1 (“IP”).

PrimaryOwnerName [Write]

Local to CIM_ComputerSystem

GenInfoSysPUser_E from the DMI General Information Group.  The provider does not support modification of this property’s value.

PrimaryOwnerContact [Write]

Local to CIM_ComputerSystem

GenInfoSysPPhone_E from the DMI General Information Group.  The provider does not support modification of this property’s value.

OtherIdentifyingInfo[]

Local to CIM_ComputerSystem

String returned by /usr/bin/model command (for example “9000/800/A180c”).

IdentifyingDescriptions[]

Local to CIM_ComputerSystem

Always returns string “Model”.
